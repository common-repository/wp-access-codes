Assigns a WordPress Role or/and a Press Permit Group to a user based on a predefined access code they enter.

== Description ==

This simple plugin enables you to set up different access codes or passwords that enable users to be added to specific WordPress Roles or  *Press Permit* (http://presspermit.com/) permission groups.

Simply go to Settings -> WP Access Codes to create and manage your codes.

Use the [wp-access-codes] shortcode to display the input field in the frontend.

**More features planned! Please make your feature requests in the support area or send a mail to info@slickorange.co.za**

== Installation ==

Does not require Press Permit - Press Permit features automatically enabled when Press Permit is installed.

1. Upload the press-permit-access-codes directory to the \`/wp-content/plugins/\` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the [pp-access-codes] shortcode to display the input field in the frontend.
